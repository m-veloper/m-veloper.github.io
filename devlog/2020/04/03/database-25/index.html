мЎ°кұҙ кІҖмғү - LIKE лҘј мқҙмҡ©н•ң кІҖмғү

  • кІҖмғү мЎ°кұҙмқ„ м •нҷ•нһҲ лӘЁлҘҙлҠ” кІҪмҡ°
  • л¶Җл¶„м ҒмңјлЎң мқјм№ҳн•ҳлҠ” лҚ°мқҙн„°лҘј кІҖмғүн•ҳл ӨлҠ” кІҪмҡ° ==> LIKE нӮӨмӣҢл“ңлҘј мӮ¬мҡ©
  • кІҖмғү мЎ°кұҙмқ„ м •нҷ•нһҲ м•Ңл©ҙ = м—°мӮ°мһҗлЎң мЎ°кұҙмқ„ н‘ңнҳ„
  • LIKE нӮӨмӣҢл“ңлҠ” л¬ёмһҗм—ҙмқ„ мқҙмҡ©н•ҳлҠ” мЎ°кұҙм—җл§Ң мӮ¬мҡ©н•  мҲҳ мһҲмқҢ

  • LIKE нӮӨмӣҢл“ңмҷҖ к°ҷмқҙ мӮ¬мҡ©н•  мҲҳ мһҲлҠ” кё°нҳё


  • LIKE нӮӨмӣҢл“ңмқҳ мӮ¬мҡ© мҳҲ

  • мҳҲм ңм—җ мӮ¬мҡ©н•  нҢҗл§Ө лҚ°мқҙн„°лІ мқҙмҠӨ

    • Customer н…Ңмқҙлё”
    • Items н…Ңмқҙлё”
    • Orders н…Ңмқҙлё”


  • Customer н…Ңмқҙлё”м—җм„ң м„ұмқҙ к№Җм”Ёмқё name, account, gradeлҘј кІҖмғүн•ҳмӢңмҳӨ.

    -SELECT name, account, grade FROM customer WHERE name LIKE вҖҳк№Җ%вҖҷ;

  • Customer н…Ңмқҙлё”м—җм„ң accountк°Җ 5мһҗмқё кі к°қмқҳ account, name, gradeлҘј кІҖмғүн•ҳмӢңмҳӨ.

    SELECT account, name, grade FROM customer WHERE account LIKE вҖҳ_ _ _ _ _вҖҷ ;

  • Customer н…Ңмқҙлё”м—җм„ң name(мқҙлҰ„)мқҳ л§Ҳм§Җл§үмһҗк°Җ вҖҳлҸҷвҖҷмқё кі к°қмқҳ name, gradeлҘј кІҖмғүн•ҳмӢңмҳӨ. ( nameмқ„ 3мһҗлЎңк°Җм •н•ң кІҪмҡ° )

    • [TIP]

      • н•ңкёҖ мҪ”л“ңк°Җ 2л°”мқҙнҠёлқјкі  н•ҳм—¬, кІҖмғүмӢң н•ңкёҖ н•ңкёҖмһҗлҘј вҖҳ_ _вҖҷмңјлЎң мӮ¬мҡ©н•ҳл©ҙ м•ҲлҗңлӢӨ.
      • н•ңкёҖ л¬ёмһҗлҸ„ л°‘мӨ„ н•ң к°ңлЎң н‘ңнҳ„н•ңлӢӨ.

    • мҳ¬л°”лҘё м§Ҳмқҳл¬ё

    • мһҳлӘ»лҗң м§Ҳмқҳл¬ё

    • SELECT name, grade FROM customer WHERE name LIKE вҖҳ_ _ _ _лҸҷвҖҷ;

  • Customer н…Ңмқҙлё”м—җм„ң name(мқҙлҰ„)мқҙ вҖҳлҸҷвҖҳмңјлЎң лҒқлӮҳлҠ” кі к°қмқҳ name, creditмқ„ кІҖмғүн•ҳмӢңмҳӨ. ( мқҙлҰ„мқҳ кёёмқҙк°Җ лӘҮ мһҗмқём§Җ м§Җм •н•  н•„мҡ”к°Җ м—ҶлҠ” кІҪмҡ° )
