SQL к°ңмҡ”


SQL (Structured Query Language)

  • лҚ°мқҙн„°лІ мқҙмҠӨм—җ лҚ°мқҙн„° мӮҪмһ…/мӮӯм ң/мҲҳм •/кІҖмғү л°©лІ•

    • кҙҖкі„ лҢҖмҲҳлӮҳ кҙҖкі„ н•ҙм„қмқ„ мӮ¬мҡ© ==> мқјл°ҳмӮ¬мҡ©мһҗк°Җ мӮ¬мҡ©н•ҳкё° м–ҙл өлӢӨ. В· лҢҖм•Ҳ : SQL мӮ¬мҡ©

  • SQL

    • кҙҖкі„ лҚ°мқҙн„°лІ мқҙмҠӨлҘј мң„н•ң н‘ңмӨҖ м§Ҳмқҳм–ҙ
    • мӮ¬мҡ©мһҗк°Җ мІҳлҰ¬лҘј мӣҗн•ҳлҠ” лҚ°мқҙн„°к°Җ л¬ҙм—Үмқём§Җл§Ң м ңмӢң
    • лҚ°мқҙн„°лҘј м–»лҠ” л°©лІ•м—җ лҢҖн•ҳм—¬ м–ёкёүн•ҳм§Җ м•ҠмқҢ
      ==> л№„м Ҳм°Ём Ғ лҚ°мқҙн„° м–ём–ҙмқҳ нҠ№м„ұ

SQLмқҳ м—ӯмӮ¬

  • 1974л…„ SEQUEL (Structured English QUEry Language)м—җм„ң мң лһҳ

    • IBM м—°кө¬мҶҢмқҳ м—°кө¬мҡ© кҙҖкі„ лҚ°мқҙн„°лІ мқҙмҠӨ кҙҖлҰ¬мӢңмҠӨн…ң (System R)мқ„ мң„н•ң м–ём–ҙ

  • мқҙнӣ„ к°ңл°ңлҗң кҙҖкі„ лҚ°мқҙн„°лІ мқҙмҠӨ кҙҖлҰ¬ мӢңмҠӨн…ңмқҖ лӢӨлҘё м§Ҳмқҳм–ҙлҘј мӮ¬мҡ©

  • 1986л…„ лҜёкөӯ н‘ңмӨҖ м—°кө¬мҶҢмқё ANSIмҷҖ көӯм ң н‘ңмӨҖнҷ” кё°кө¬мқё ISOм—җм„ң SQLмқ„ кҙҖкі„ лҚ°мқҙн„°лІ мқҙмҠӨмқҳ н‘ңмӨҖ м§Ҳмқҳм–ҙлЎң мұ„нғқн•ҳкі  н‘ңмӨҖнҷ” мһ‘м—… м§„н–ү

  • 1986л…„ н‘ңмӨҖ SQL : SQL-86 лҳҗлҠ” SQL1

  • 1992л…„ н‘ңмӨҖ SQL : SQL-92 лҳҗлҠ” SQL2

  • 1999л…„ н‘ңмӨҖ SQL : SQL-99 лҳҗлҠ” SQL3

SQL мӮ¬мҡ© л°©лІ•

  • DBMS лӮҙм—җм„ң лҢҖнҷ”мӢқмңјлЎң м§Ҳмқҳ
  • C, C++, JavaмҷҖ к°ҷмқҖ м–ём–ҙлЎң мһ‘м„ұн•ң н”„лЎңк·ёлһЁм—җ мӮҪмһ…н•ҳм—¬ мӮ¬мҡ©

SQLмқҳ л¶„лҘҳ

  • лҚ°мқҙн„°м •мқҳм–ҙ (DDL, Data Definition Language)

    • н…Ңмқҙлё”мқ„ мғқм„ұ, ліҖкІҪ, мӮӯм ңн•ҳлҠ” кё°лҠҘ

  • лҚ°мқҙн„°мЎ°мһ‘м–ҙ (DML, Data Manipulation Language)

    • н…Ңмқҙлё”м—җ лҚ°мқҙн„°лҘј мӮҪмһ…, мҲҳм •, мӮӯм ң, кІҖмғүн•ҳлҠ” кё°лҠҘ

  • лҚ°мқҙн„°м ңм–ҙм–ҙ (DCL, Data Control Language)

    • ліҙм•Ҳмқ„ мң„н•ҙ лҚ°мқҙн„°м—җ лҢҖн•ң м ‘к·ј л°Ҹ мӮ¬мҡ© к¶Ңн•ңмқ„ мӮ¬мҡ©мһҗлі„лЎң л¶Җм—¬, м·ЁмҶҢн•ҳлҠ” кё°лҠҘ

    • лҚ°мқҙн„°лІ мқҙмҠӨ кҙҖлҰ¬мһҗк°Җ мЈјлЎң мӮ¬мҡ©

SQLмқҳ кё°лҠҘ

  • мЈјкё°лҠҘ : кІҖмғүмқ„ мң„н•ң м§Ҳмқҳ мһ‘м„ұмҡ© л№„м Ҳм°Ём Ғ лҚ°мқҙн„° мЎ°мһ‘м–ҙ

  • ліҙмЎ°кё°лҠҘ : лҚ°мқҙн„° м •мқҳ л°Ҹ м ңм–ҙ

MySQL н…Ңмқҙлё” мғқм„ұ, AUTO_INCREMENT(мһҗлҸҷмҰқк°Җ)


мӢӨмҠөмҡ© лҚ°мқҙн„°лІ мқҙмҠӨ

  • A. Customer н…Ңмқҙлё”

  • B. Items н…Ңмқҙлё”

  • C. Orders н…Ңмқҙлё”

н…Ңмқҙлё” мғқм„ұ вҖ“ мҶҚм„ұ м •мқҳ


  • A. н…Ңмқҙлё” мғқм„ұмӢң м •мқҳн•  мӮ¬н•ӯ

    • в‘  н…Ңмқҙлё” мқҙлҰ„
    • в‘Ў мҶҚм„ұ мқҙлҰ„ л°Ҹ лҚ°мқҙн„° нғҖмһ…, м ңм•Ҫ мӮ¬н•ӯ
    • в‘ў кё°ліёнӮӨ, лҢҖмІҙнӮӨ, мҷёлһҳнӮӨмқҳ м •мқҳ
    • в‘Ј лҚ°мқҙн„° л¬ҙкІ°м„ұмқ„ мң„н•ң м ңм•ҪмЎ°кұҙ м •мқҳ


  • SQL м§Ҳмқҳл¬ё

    • м„ёлҜёмҪңлЎ ( ; )мңјлЎң л¬ёмһҘмқҳ лҒқмқ„ н‘ңмӢң
    • лҢҖмҶҢл¬ёмһҗ кө¬л¶„н•ҳм§Җ м•ҠмқҢ


  • B. мҶҚм„ұ м •мқҳ

    • н…Ңмқҙлё”мқ„ кө¬м„ұн•ҳлҠ” мҶҚм„ұ м •мқҳ

      • лҚ°мқҙн„° нғҖмһ… м •мқҳ
      • NULL к°’ н—Ҳмҡ© м—¬л¶Җ м •мқҳ
      • кё°ліё к°’ н•„мҡ” м—¬л¶Җ


  • C. мҶҚм„ұмқҳ лҚ°мқҙн„° нғҖмһ…

  • D. NOT NULL

    • мҶҚм„ұмқҳ NULL к°’мқ„ н—Ҳмҡ©н•ҳм§Җ м•ҠмқҢмқ„ мқҳлҜё
    • л°ҳл“ңмӢң к°’мқҙ мһ…л Ҙлҗҳм–ҙм•ј н•ҳлҠ” мҶҚм„ұм—җ мӮ¬мҡ©
    • мқён„°л„· мғҒмқҳ нҡҢмӣҗк°Җмһ…мӢң н•„мҲҳн•ӯлӘ©
    • мҳҲ : account VARCHAR(20) NOT NULL


  • E. DEFAULT

    • мҶҚм„ұмқҳ кё°ліё к°’мқ„ м§Җм •
    • DEFAULTлҘј м§Җм •н•ҳм§Җ м•Ҡкі , к°’мқҙ мһ…л Ҙлҗҳм§Җ м•ҠмқҖ кІҪмҡ° NULLк°’мңјлЎң мһҗлҸҷ м ҖмһҘ
    • мҳҲ : credit INT DEFAULT 0
    • мҳҲ : grade VARCHAR(10) DEFAULT вҖҳNEWвҖҷ

    [TIP]

    • л¬ёмһҗм—ҙ, лӮ м§ң лҚ°мқҙн„° : мһ‘мқҖ л”°мҳҙн‘ңлЎң л¬¶м–ҙм•ј н•Ё
    • мһ‘мқҖ л”°мҳҙн‘ң лӮҙмқҳ л¬ёмһҗм—ҙ : лҢҖмҶҢл¬ёмһҗ кө¬л¶„лҗЁ

      • мҳҲ :

        • grade VARCHAR(10) DEFAULT вҖҳNEWвҖҷ
        • grade VARCHAR(10) DEFAULT вҖҳNewвҖҷ


  • F. AUTO_INCREMENT

    • н…Ңмқҙлё”м—җ мғҲлЎңмҡҙ нҲ¬н”Ңмқҙ мӮҪмһ…лҗ  л•Ң, мһҗлҸҷмңјлЎң DBMSм—җм„ң мң мқјн•ң лІҲнҳёлҘј мғқм„ұ
    • AUTO_INCREMENTмқё мҶҚм„ұмқҖ кё°ліёнӮӨлЎң м§Җм •лҗҳм–ҙм•ј н•Ё.
    • 1л¶Җн„° мӢңмһ‘н•ҳл©°, мғҲлЎңмҡҙ нҲ¬н”Ң(л ҲмҪ”л“ң) м¶”к°ҖмӢң 1м”© мҰқк°Җ
    • нҳ•мӢқ : мҶҚм„ұмқҙлҰ„ INT NOT NULL AUTO_INCREMENT

    • лӢӨлҘё к°’л¶Җн„° мӢңмһ‘н•ҳлҸ„лЎқ м„Өм •н•  мҲҳ мһҲлӢӨ.

      • нҳ•мӢқ : ALTER TABLE н…Ңмқҙлё”лӘ… AUTO_INCREMENT=мҙҲкё°к°’
    • мғҲлЎңмҡҙ л ҲмҪ”л“ңлҘј мӮҪмһ…н•  л•Ң, AUTO_INCREMENTлЎң м§Җм •лҗң мҶҚм„ұк°’мқҖ мһ…л Ҙн•ҳм§Җ м•Ҡм•„лҸ„ лҗЁ
    • AUTO_INCREMENTлЎң м§Җм •лҗң мҶҚм„ұмқҳ нҳ„мһ¬ к°’мқҖ ліҖкІҪн•  мҲҳ м—ҶмқҢ

    • л ҲмҪ”л“ң мӮҪмһ…мӢң SQL кө¬мЎ°

      • INSERT INTO н…Ңмқҙлё”лӘ… ( мҶҚм„ұ1, вҖҰ мҶҚм„ұN ) values ( мҶҚм„ұк°’1, вҖҰ мҶҚм„ұк°’N )
      • AUTO_INCREMENTлЎң м§Җм •лҗң мҶҚм„ұмқҖ м ңмҷён•  мҲҳ мһҲмқҢ

  • G. мӢӨмҠөмҡ© лҚ°мқҙн„°лІ мқҙмҠӨ лҚ°мқҙн„° м •мқҳ

    • Customer н…Ңмқҙлё”

мӢӨмҠөмҡ© лҚ°мқҙн„°лІ мқҙмҠӨ лҚ°мқҙн„° м •мқҳ (Customer н…Ңмқҙлё”)

  • кё°ліё н…Ңмқҙлё” мғқм„ұ

  • лҚ°мқҙн„° мһ…л Ҙ

  • лҚ°мқҙн„°мһ…л Ҙ : NOT NULL мҶҚм„ұ лҚ°мқҙн„°л§Ң мһ…л ҘмӢң

  • AUTO_INCREMENT м Ғмҡ©мӢң

  • AUTO_INCREMENT мқҳ мҙҲкё°к°’мқ„ 100мңјлЎң м„Өм •н•ҳкё°

    • н…Ңмқҙлё” мғқм„ұ м§Ғнӣ„м—җ м„Өм •

  • AUTO_INCREMENT м Ғмҡ©мӢң INSERT мҳӨлҘҳ

  • AUTO_INCREMENT : мҶҚм„ұмқҙлҰ„ лҰ¬мҠӨнҠёлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкі  мһ…л Ҙн•ҳкё°

    • AUTO_INCREMENT мҶҚм„ұ к°’ = мҲ«мһҗ мқҙмҷёмқҳ лӘЁл“  л¬ёмһҗм—ҙмқҙлӮҳ NULLмқ„ мҶҚм„ұ к°’мңјлЎң мһ…л Ҙн•ҳл©ҙ DBMSлҠ” мһҗлҸҷмңјлЎң лӢӨмқҢ мҲ«мһҗлҘј мӮ¬мҡ©н•ҳм—¬ нҲ¬н”Ңмқ„ мӮҪмһ…н•Ё

    • AUTO_INCREMENT мҶҚм„ұ к°’ = мҲ«мһҗлҘј мӮ¬мҡ©н•ҳл©ҙ, мһ…л Ҙн•ң мҲ«мһҗлЎң к°’мқҙ м§Җм •лҗЁ
