Sass(SCSS) к°Җм ёмҳӨкё°(Import)


  • @importлЎң мҷёл¶Җм—җм„ң к°Җм ёмҳЁ Sass нҢҢмқјмқҖ лӘЁл‘җ лӢЁмқј CSS м¶ңл Ҙ нҢҢмқјлЎң лі‘н•©лҗ©лӢҲлӢӨ.
  • к°Җм ёмҳЁ нҢҢмқјм—җ м •мқҳлҗң лӘЁл“  ліҖмҲҳ лҳҗлҠ” Mixins л“ұмқ„ мЈј нҢҢмқјм—җм„ң мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • Sass @importлҠ” кё°ліём ҒмңјлЎң Sass нҢҢмқјмқ„ к°Җм ёмҳӨлҠ”лҚ°, CSS @import к·ңм№ҷмңјлЎң м»ҙнҢҢмқјлҗҳлҠ” лӘҮ к°Җм§Җ мғҒнҷ©мқҙ мһҲмҠөлӢҲлӢӨ.

    • нҢҢмқј нҷ•мһҘмһҗк°Җ .cssмқј л•Ң
    • нҢҢмқј мқҙлҰ„мқҙ http://лЎң мӢңмһ‘н•ҳлҠ” кІҪмҡ°
    • url()мқҙ л¶ҷм—Ҳмқ„ кІҪмҡ°
    • лҜёл””м–ҙмҝјлҰ¬к°Җ мһҲлҠ” кІҪмҡ°
@import "hello.css";
@import "http://hello.com/hello";
@import url(hello);
@import "hello" screen;


м—¬лҹ¬ нҢҢмқј к°Җм ёмҳӨкё°

  • н•ҳлӮҳмқҳ @importлЎң м—¬лҹ¬ нҢҢмқјмқ„ к°Җм ёмҳ¬ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. нҢҢмқј мқҙлҰ„мқҖ ,лЎң кө¬л¶„н•©лӢҲлӢӨ.
@import "header", "footer";


нҢҢмқј л¶„н• (Partials)

  • н”„лЎңм қнҠё к·ңлӘЁк°Җ м»Өм§Җл©ҙ нҢҢмқјл“Өмқ„ headerлӮҳ side-menu к°ҷмқҙ к°Ғ кё°лҠҘкіј л¶Җл¶„мңјлЎң лӮҳлҲ  мң м§ҖліҙмҲҳк°Җ мүҪлҸ„лЎқ кҙҖлҰ¬н•ҳкІҢ лҗ©лӢҲлӢӨ.

  • мқҙ кІҪмҡ° нҢҢмқјмқҙ м җм җ л§Һм•„м§ҖлҠ”лҚ°, лӘЁл“  нҢҢмқјмқҙ м»ҙнҢҢмқј мӢң к°Ғк°Ғмқҳ ~.css нҢҢмқјлЎң лӮҳлҲ м„ң м ҖмһҘлҗңлӢӨл©ҙ кҙҖлҰ¬лӮҳ м„ұлҠҘ м°Ёмӣҗм—җм„ң л¬ём ңк°Җ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • к·ёлһҳм„ң SassлҠ” Partials кё°лҠҘмқ„ м§Җмӣҗн•©лӢҲлӢӨ.

  • нҢҢмқј мқҙлҰ„ м•һм—җ _лҘј л¶ҷм—¬(_header.scssмҷҖ к°ҷмқҙ) @importлЎң к°Җм ёмҳӨл©ҙ м»ҙнҢҢмқј мӢң ~.css нҢҢмқјлЎң м»ҙнҢҢмқјн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

  • мҳҲлҘј л“Өм–ҙліҙкІ мҠөлӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҙ scss/ м•Ҳм—җ 3к°ңмқҳ Sass нҢҢмқјмқҙ мһҲмҠөлӢҲлӢӨ.

// main.scss
@import "header", "side-menu";
  • к·ёлҰ¬кі  мқҙ нҢҢмқјл“Өмқ„ css/л””л үнҶ лҰ¬лЎң м»ҙнҢҢмқјн•©лӢҲлӢӨ. (м»ҙнҢҢмқјмқҖ node-sass)
# `scss`л””л үнҶ лҰ¬м—җ мһҲлҠ” нҢҢмқјл“Өмқ„ `css`л””л үнҶ лҰ¬лЎң м»ҙнҢҢмқј
$ node-sass scss --output css
